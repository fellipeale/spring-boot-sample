#Runbook
Guide to install and run the application.

##Prerequisites
* Maven
* Node.js
* GIT

##Installation
Run those commands in the terminal:

    git clone https://github.com/fellipeale/spring-boot-sample.git
    
    cd spring-boot-sample

    mvn install

    npm install

##Running
Run those commands in the terminal:

    mvn spring-boot:run &

    npm start &

Go to http://localhost:3000/
